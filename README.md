# discord-api-lib
A discord library made for easily interacting with discord's API in order to mess around with a user.

Currently supported features
----------------------------
GET:
    NewHeaders
    RandomNonce
    RandomCookies
    PaymentMethods
    UserChannelID
    Name
    Bio
    Id
    Email
SET:
    Email
    Name
    Bio
ACTIONS:
    MakeRequest
    AddFriend
    RemoveFriend
    SendMessage
    ScrapeMessages
    PurgeMessages
    JoinGuild
